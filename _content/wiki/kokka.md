# 🏰 国家鯖機能 総合ガイド

このガイドでは、当サーバーで国家運営や生活を楽しむために必要な主要コマンドをまとめています。

---

## 🏗️ Towny（土地保護・国家）
町を作り、国を発展させるための根幹システムです。
詳細な使い方は **[Towny解説Wiki](https://nami-minecraft.vercel.app/wiki/towny)** を確認してください。

| コマンド | 説明 |
| :--- | :--- |
| `/t new <町名>` | 今立っている場所を中心に新しい町を作成します。<br>**※町名は「半角英数字」のみ使用可能です。** |
| `/t deposit <金額>` | 町の銀行にお金を寄付します（維持費に必要）。 |
| `/t claim` | 今立っているチャンクを町の領土として保護します。 |
| `/t spawn` | 所属している町のスポーン地点へ移動します。 |
| `/n new <国名>` | 自分の町を首都として新しい国を建国します。 |

## 🧳 基本・移動コマンド
拠点の設定や他プレイヤーへのテレポートなど、日常的に使うコマンドです。
詳しくは **[コマンド一覧Wiki](https://nami-minecraft.vercel.app/wiki/commands)** を参照してください。

| コマンド | 説明 |
| :--- | :--- |
| `/sethome [名前]` | 今立っている場所をホーム（拠点）として設定します。 |
| `/home [名前]` | 設定したホームへテレポートします。 |
| `/tpa <プレイヤー名>` | 特定のプレイヤーにテレポート申請を送ります。 |
| `/tpaccept` | 送られてきたテレポート申請を承認します。 |
| `/spawn` | サーバーのメインスポーン地点へ戻ります。 |

---

## 🔒 Bolt（チェスト・看板保護）
チェストやドアを保護し、盗難を防ぐシステムです。
詳細な権限設定は **[Bolt解説Wiki](https://nami-minecraft.vercel.app/wiki/bolt)** を確認してください。

| コマンド | 説明 |
| :--- | :--- |
| `/bolt lock` | チェストなどを自分専用にロックします（設置時に自動ロックされます）。 |
| `/bolt unlock` | ロックを解除します。 |
| `/bolt modify +<名前>` | 指定したプレイヤーに操作権限を追加します。 |
| `/bolt info` | 保護の状態や権限を持っているプレイヤーを確認します。 |

---

## 💎 DiamondBank（銀行）
ダイヤモンドを電子データとして安全に預けられるシステムです。

| コマンド | 説明 |
| :--- | :--- |
| `/deposit <枚数>` | 手持ちのダイヤモンドを銀行に預けます。 |
| `/withdraw <枚数>` | 銀行からダイヤモンドをアイテムとして引き出します。 |
| `/bal` | 自分の銀行残高を確認します。 |
| `/bank top` | サーバー内の富豪ランキングを表示します。 |

### 💡 国家鯖で生き残るためのヒント
* **資産の保護:** チェストを置いたら `/bolt info` で保護を確認し、ダイヤは `/deposit` で銀行に預けるのが最も安全です。
* **町名・国名のルール:** 漢字やひらがなは使えません。必ず半角英数字で命名してください。
* **維持費の重要性:** 町の銀行残高がなくなると、土地の保護が消滅します。定期的に `/t deposit` しましょう。
